Drink, Drank, Drunk

Abstract

Drink, Drank, Drunk is a Blood Alcohol Content calculator on your smart phone! Drink, Drank, Drunk will display estimated blood alcohol content (BAC) level,along with an intoxication graph and possible next steps for the user. Simply enter in personal information including age and weight, and number of drinks to receive results. In order for this application to run smoothly, a number of development technologies, frameworks, languages and libraries need to used.
Demographic
• Adults in their late 20’s to early 40’s
• Most likely to utilize the application
• Likely to own vehicles, rather than students
• Enjoy going out socially for drinks • Responsible
• Concern about driving infractions

Sponser

The application is designed to prevent and discourage drunk driving by providing the user with their blood alcohol content (BAC). Driving while under the influence is 100% preventable, and Drink, Drank, Drunk is another tool for responsible drivers to use to further prevent driving drunk. Having an application right on driver’s phones is convenient and will eliminate all excuses.
Drink, Drank, Drunk has been sponsored by one of London's most popular and longest running desigated driver services, Diamondz DD.

Features

• Estimate user's blood alcohol content based on weight, gender, and drinks consumed

• Add drink to user's session from pre-existing list

• Ability to input custom drink to user's session

• Display user's current blood alcohol content

• Display user's blood alcohol content on graph

• Ability to choose local car services based on the user's geo-location

• App sponsor displays first on car services option

Tech Specs

Drink, Drank, Drunk will be a primarily mobile application, because the ideal user will be out of the house, away from a computer, enjoying their night. This ideal user is smart phone savvy, and can easily pull up the application on their smart phone device, rather than need a computer. In order to create this application I have used technologies including Foundation 5 grid framework, Codeigniter MVC framework, and PhoneGap.

Java-script has been used to create the front-end for the application. I have used Java-script to control the equations to calculate the user’s blood alcohol content, with decreasing time from consumption. In order to display the user’s Blood Alcohol Content I used Chart.js a Java-script library.

The back-end approach have taken is using an MVC pattern and a PHP framework, Codeigniter. The back-end stores data including user login, user profile, and alcohol information.

In order to get Drink, Drank, Drunk onto the apple and android store I have wrapped the application with PhoneGap. PhoneGap is an open source framework for building cross-platform mobile applications using HTML, CSS, and JavaScript. Using PhoneGap will eliminate the need to write in native development languages, specific to each platforms operating system. Once the application is wrapped, it will be able to be placed in multiple app stores ( iTunes app store, Google play store, etc.).
